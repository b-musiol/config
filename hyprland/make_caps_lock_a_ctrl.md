* Using KoolDots (v2.3.22)

To change Caps Lock to CTRL:
1. Hit `SHIFT+SUPER+E`
2. Select `Edit User Settings`
3. Scroll to `input`
4. Add `caps:ctrl_modifier` to `kb_options`. If something exists there, append with a comma (`,`).
5. Save the config file.
6. Close the config file.
7. hyprland automatically uses this now.
