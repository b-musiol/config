To set up a good hyprland starting point:
1. Make a folder somewhere
2. Go to that folder in the terminal
3. Run `sh <(curl -L https://raw.githubusercontent.com/LinuxBeginnings/Hyprland-Dots/main/Distro-Hyprland.sh)` (this is from `https://github.com/LinuxBeginnings/Hyprland-Dots`)
4. run `./install.sh`

If you run this from CachyOS and preinstall hyprland, just ignore the first steps suggested there.

If you can deal with using a US keyboard layout for a second, don't change it. You get to change it within the KoolDots installations script.
