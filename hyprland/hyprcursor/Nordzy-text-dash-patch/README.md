This is a patch for the regular Nordzy-hyprcursors pack
It adds a dash in the middle of the text cursor. This makes aiming at texts with large cursors easier.

Duplicate the Nordzy-hyprcursors pack and replace the `text.hlc` file.

You might need to change the `mainfest.hl` to a different name to indicate that this is the patched version. Alternatively patch the original files (not recommended).
