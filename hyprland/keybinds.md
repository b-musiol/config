This is a collection of keybinds I use. They are intended for use in hyprland, but also work for Niri if an equivalent exists there.

This is not a setup script. This is a reference.

# Keybinds

Keybind                        | Function
-------------------------------|--------------------------------------
SUPER+Q                        | Exit current window
SUPER+CTRL+Q                   | Kill current window
SUPER+E                        | Open File Manager
SUPER+Return                   | Open Terminal
SUPER+D                        | Open Application Launcher
SUPER+B                        | Open Browser
SUPER+*n*                      | Go to workspace *n*
SUPER+CTRL+*n*                 | Move current window to workspace *n*
SUPER+*arrow_key*              | Switch window focus
SUPER+CTRL+*arrow_key*         | Swap window position
SUPER+ALT+E                    | Emoji selector
SUPER+H                        | Shortcut cheat sheet
